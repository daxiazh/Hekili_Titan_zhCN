## 核心机制理解

### 1. 锁定装填 (Lock and Load) ⭐⭐⭐

这是生存猎人最核心的机制！

**触发条件:**
- 黑箭每次跳伤害时有几率触发
- 冰冻陷阱、爆炸陷阱触发时有几率触发

**效果:**
- 下2次爆炸射击无CD、无消耗
- 持续12秒

**使用技巧:**
```
锁定装填触发后:
爆炸射击 → 等待1秒(避免DoT覆盖) → 爆炸射击
```

⚠️ **重要**: 两发爆炸射击之间必须间隔约1秒，否则第二发会覆盖第一发的DoT伤害！

### 2. 改进稳固射击

**触发条件:**
- 每次稳固射击有15%几率触发

**效果:**
- 下一个瞄准射击/奥术射击/奇美拉射击伤害+15%
- 法力消耗-20%

**使用技巧:**
- 触发后优先使用瞄准射击消耗buff

### 3. 狙击训练

**效果:**
- 站桩超过6秒后，伤害+6%

**使用技巧:**
- 尽量减少移动，保持站桩输出
- 移动时使用即时技能

## 实战技巧

### 1. 锁定装填最大化

- **保持黑箭100%覆盖率**: 黑箭是触发锁定装填的主要来源
- **爆炸射击间隔**: 锁定装填触发后，两发爆炸射击之间等待约1秒
- **不要浪费**: 锁定装填持续12秒，确保在消失前用完

### 2. 移动输出

生存猎人移动时可用的即时技能:
- 黑箭 (即时)
- 毒蛇钉刺 (即时)
- 爆炸射击 (即时)
- 杀戮射击 (即时)
- 瞄准射击 (即时)

**技巧**: 移动时优先刷新DoT，站桩时使用稳固射击

### 3. 法力管理

- **狩猎派对**: 暴击时回复法力，保持高暴击率
- **蝰蛇守护**: 法力<20%时切换，>80%时切回龙鹰
- **稳固射击**: 本身也能回复少量法力

### 4. 爆发窗口

最佳爆发时机:
```
急速射击 + 饰品 + 嗜血/英勇 → 疯狂输出
```

在爆发窗口内:
- 确保黑箭和毒蛇钉刺已经挂上
- 优先使用爆炸射击
- 锁定装填触发时立即消耗

### 5. 站桩输出

- 狙击训练需要站桩6秒才能获得+6%伤害
- 尽量预判Boss技能，提前移动到安全位置
- 移动后尽快恢复站桩

## 常见问题

### Q: 锁定装填触发后应该怎么打?

A: 
```
爆炸射击 → 稳固射击(或等待1秒) → 爆炸射击
```
不要连续两发爆炸射击，会覆盖DoT！

### Q: 黑箭和毒蛇钉刺哪个优先?

A: 黑箭优先。黑箭能触发锁定装填，是DPS的关键。

### Q: 什么时候用瞄准射击?

A: 只在有"改进稳固射击"buff时使用，否则DPS不如稳固射击。

### Q: AOE时还需要维护DoT吗?

A: 3+目标时优先AOE技能，但如果有高血量目标(如Boss)，仍需维护黑箭。
